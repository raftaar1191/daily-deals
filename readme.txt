Provided a new tab in CPT Product admin section and a tab to set the product deals timing.

== Description ==

This plugin provided a new tab in product single page admin section where user can set the prices an quantity of the product and a tab to set the product deals timing

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/deal-deals-by-harimay` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Products->Daily Deals Setting screen to configure the plugin
